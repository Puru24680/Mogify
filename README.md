# 🚀 Mogify

Mogify is an AI-powered facial analysis platform that helps users better understand their facial features and receive personalized appearance improvement recommendations.

> ⚠️ This project is currently under active development.

---

## ✨ Features

- 📸 Upload face images
- 🤖 AI-powered facial analysis
- 📊 Facial metrics dashboard
- 💇 Hairstyle recommendations
- 🧴 Skincare & grooming suggestions
- 📈 Progress tracking
- 🔐 Secure authentication
- ☁️ Cloud image storage

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Puru24680/mogify.git
```

### Install frontend dependencies

```bash
cd client
npm install
```

### Install backend dependencies

```bash
cd ../server
npm install
```
